# Pabliki Client Radar — интерактивный прототип

Next.js 14 (App Router) + TailwindCSS. Демо-данные, без бэкенда — это UI-скелет дашборда
для демонстрации интерфейса и навигации.

## Запуск

```bash
cd prototype
npm install
npm run dev      # http://localhost:3000
```

Сборка прод-версии: `npm run build && npm run start`.

## Что внутри

| Маршрут | Экран |
|---|---|
| `/` | Dashboard (метрики, топ лиды, задачи, ниши/города/источники, воронка) |
| `/leads` | Таблица лидов с фильтрами (приоритет/город/ниша/поиск) |
| `/leads/[id]` | Карточка лида (3 колонки: идентификация · сигнал+score · действия+сообщения) |
| `/signals` | Лента сигналов |
| `/sources` | Источники (11 коннекторов, статус, режим) |
| `/tasks` | Канбан задач менеджеров |
| `/messages` | Библиотека сгенерированных сообщений (9 типов) |
| `/import` | Импорт Excel/CSV (мастер из 5 шагов) |
| `/analytics` | Расширенная аналитика |
| `/settings` | Команда, роли, API-ключи, webhooks, веса Priority Score |

## Брендинг

Светлая тема, синий акцент `#2563EB`, графит `#1F2937`, белые карточки. Цвета приоритетов:
красный (90–100), жёлтый/янтарный (75–89), голубой (50–74), серый (<50).
